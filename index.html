自己紹介 About me

はじめまして、長谷川 拓音 と申します。武蔵野大学データサイエンス学部で、データ分析と Web 開発をいったりきたりしている 3 年生です。

ふだんは渡邉ゼミで 「スキルギャップレーダー」 という、企業の人材運用を効率化するための Web アプリを作っています。バックエンドとフロントエンド両方を触るのが好きで、最近はそこに LLM を組み込んだプロダクトづくりに夢中です。

モットーは 「迷ったら、行動する。」。コードが行き詰まったら、お猪口で一杯だけ頂きながら考えることにしています。
